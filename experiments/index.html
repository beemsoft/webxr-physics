WebXR Physics: Experiments

  • Basic
    This is a basic demo of a cube in zero-gravity space. The cube emits a sound that is played using the Resonance Audio SDK for Web.
  • Backspin
    A basketball is dropped from the sky. You can grab it and throw it away. The demo was created for testing the backspin effect on the ball. A texture loader is used in order to display a realistic basketball.
  • Arm model test
    This is the arm model test from the Ray-input project from Boris Smus. It was used for doing tests with a single controller device for the Oculus Go. However, since the arrival of the Oculus Quest, the arm model is not needed any more.
  • Ragdoll
    This experiment lets your body take control over a rag doll. Your head and both hand positions will be used for moving the rag doll. Both feet are still fixed to the ground.
  • Basic turn
    This experiment is using two rag dolls. One is controlled by your head and hand positions and can move around. The other one is fixed to the ground and can be turned around.
  • Ragdoll with ball
    This experiment features a rag doll and a basketball.
